WOW BITES 🍔
Burger Brand Identity & Promotional Design Project

WOW BITES is a bold and energetic fast-food brand concept created for a burger outlet.
The project focuses on building a visually appealing restaurant brand identity through logo design, menu design, and promotional food posters.
